This is a modified version of the Panda dataset.
The folder contains:
- code: code and config to reproduce the results
- patches: the patch images
- masks: the label masks with colour: original_label*50 for each pixel
- train_patches.csv: table of all patch labels of the training split
- test_patches.csv: table of all patch labels of the test split
- val_patches.csv: table of all patch labels of the val split
